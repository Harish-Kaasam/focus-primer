# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-15

### Added
- Initial release of the **Focus Primer** desktop application.
- Desktop UI matching *Headspace*, *Arc*, and *Apple Fitness* layout aesthetics.
- Interactive **Screen 1 (Setup)**, **Screen 2 (Fixation)**, **Screen 3 (Ujjayi Breathing)**, and **Screen 4 (Focus Session)**.
- Local statistics persistence storage via JSON file.
- Cross-platform dynamic sine-wave beep generators.
- Built installer script with desktop and start menu shortcut links.
