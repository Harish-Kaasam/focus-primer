# Roadmap

We plan to expand Focus Primer to support broader features:

- **Custom Themes**: Integrate light luxury themes and user-definable custom hex configurations.
- **Sound Packs**: Offer additional chime presets (gongs, bowls, ocean waves).
- **Streak History View**: Integrate graphical visual charts showing stats.
- **Settings Panel**: Allow fine-tuning UI transitions, hardware acceleration configurations, and timer settings.
- **Multiplatform Builds**: Create verified workflows for macOS DMG and Linux binaries.
